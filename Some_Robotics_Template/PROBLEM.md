# Robotics Utility Toolkit

Create a generic robotics utility toolkit in C++ using templates. Implement a `DataHolder<T>` template class capable of storing a value of any type, such as `double` or `string`. Add reusable function templates including `midpoint()` to compute the average of two numeric values and `swapValues()` to exchange any two variables. Demonstrate the toolkit by computing robot-relevant constants and swapping robot component labels, showcasing how C++ templates enable type-safe, reusable code.
