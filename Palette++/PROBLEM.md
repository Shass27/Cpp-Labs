# Palette++

Implement a `Color` class in C++ that represents an RGB color with three integer channels clamped to the valid 0–255 range. Support two static factory methods — `fromPackedRgb` to construct a color from a single 24-bit integer and `fromHexLiteral` to parse a CSS-style hex string such as `"#ff4500"`. Practice operator overloading by implementing channel-wise color mixing with `operator+`, per-channel adjustment via a `std::vector<int>` overload, increment and decrement operators that shift every channel by one, scalar multiplication with `operator*`, a function-call operator that applies an arbitrary per-channel transform (e.g. `ColorUtility::invert`), equality comparison, and stream insertion/extraction operators. Implement the companion `ColorUtility` class with static helpers for hex-digit conversion and channel clamping.
